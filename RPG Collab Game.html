RPG Collab

This is the project of the module named:Junior/Senior Collaborative Game Development and Testing

My role in this Game Jam

In this project I will be the Leader of the project, have the project management although I will do every part of the project include Game Designer, Programmer and a little UI/UX.

Game Name: Astral Fanatic

Astral Fanatic is the RPG game which player will join as a soldier that will have their team to compete with the enemy in the game and do the quest of the game to finish the game. Player can upgrade level of the team, equip weapon to make the team better.

Work Detail

I did the Item system of scriptable object which make the system easier. Then I help the team to finish the Inventory system which need to add item into the inventory and make the item show to the UI.Then I made the Quest System that is the main system in this game and the quick shopping system.
